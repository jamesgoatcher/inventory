# inventory

I have a lot of interests that help define me and many of them manifest as physical goods. I’m pretty ridiculous about keeping track of everything in my life so I built this to help ease my mind. Behold - the ultimate vanity project: Inventory! I have catalogued my dearest positions and allow the world to access them and my commentary in a Mac OS finder inspired browser window. Written in ECMA6 while also showcasing my UI.
